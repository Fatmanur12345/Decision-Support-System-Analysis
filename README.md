# 📊 Strategic Decision Support System (DSS) Simulation

This research-oriented project focuses on the interaction between **Managerial Decision-Making** and **Real-Time Data Analysis** in operational environments.

## 🔍 Research Context
- **Human-Technology Interaction:** Analyzing how visual data representation assists managers in rapid capacity adjustments.
- **Scenario Analysis:** Exploring the effects of "Market Conditions" (e.g., Holiday season, Economic recession) on pricing and personnel allocation.

## 🚀 Key Features
- **Dynamic Risk Engine:** Hybrid risk estimation based on historical cancellation rates and real-time "no-show" probabilities.
- **Automated Audit Engine:** An AI-inspired reporting layer that triggers alerts based on strategic imbalances (e.g., excessive discounting or personnel shortages).
- **Interactive Dashboards:** Visualized metrics for capacity utilization, expected net profit, and resource management.

## 🛠️ Technical Stack
- **Python / Streamlit**
- **Pandas & NumPy** (For deterministic data simulation)
- **Plotly** (For interactive visualization)
